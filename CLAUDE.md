# Project Overiew

Canvas Swiss-Army Knife

This commandline tool helps manage canvas course content from the commandline

# Commands

subcommands have their own python file in canvas_sak/commands

# Due dates files

- each assignment is on its own line
- the date format is YYYY-MM-DD-hh:mm
- each line assignment name tab comma separated dates where each data starts with type (available, due, util) then = then the date

# dryrun

for commands that modify things add a --no-dryrun option that will apply the change. --dryrun (the default) will only display what would happen

# bug fixes

1. reproduce the bug
2. add a test case before fixing the bug
3. fix the bug
4. verify the test case passes

# features

1. add a test case for the feature
2. add the feature
3. verify the test case passes
